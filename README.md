# AntOA
本项目是一个基于Vue Antd Admin[Vue Antd Admin](https://github.com/iczer/vue-antd-admin) 大刀阔斧地阉割并二次开发的面向 Laravel 后台开发者的后台管理系统框架，laravel开发者可以通过简单的配置轻松实现各种后台简单或稍显复杂的交互
